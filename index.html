BlackJack game   

Button: depending on the button pushed, specific operation will be called to play the game

Output: string: dealers hand, string : players hand, String: outcome of game

 no challenging http://webstrar82.fulton.asu.edu/page0/ This is a WSDL service. Create a service to play a game of blackjack using the users input from a button push and display the cards and outcome(can continue, lost, win). Uses three operationsContracts (DealCards(), Play(string, string) and Cards()). All read and write to files depending on the operation. Write my own code and used multiple files to save information for player, dealer, deck, reset deck.
